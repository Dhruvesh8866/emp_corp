1st Step: Extract file
2nd Step: Copy the main project folder(employee_management)
3rd Step: Paste in xampp/htdocs/
4th Step: Open a browser and go to URL “http://localhost/phpmyadmin/”
5th Step: Then, click on the databases tab
6th Step: Create a database naming “370project” and then click on the import tab
7th Step: Click on browse file and select “370project.sql” file which is inside the “employee_management” folder
8th Step: Click on go.

After Creating Database,

9th Step: Open a browser and go to URL “http://localhost/employee_management/”
